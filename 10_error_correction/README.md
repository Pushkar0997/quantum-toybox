# Error Correction

This folder contains small error-correction experiments that illustrate how simple correction circuits respond to bit-flip and phase-flip style errors.

## Notebooks

- [3_qubit_bit_flip.ipynb](3_qubit_bit_flip.ipynb) demonstrates a three-qubit bit-flip correction scheme.
- [phase_flip_experiments.ipynb](phase_flip_experiments.ipynb) explores phase-flip behavior and correction ideas.

## What to Expect

- Encoded logical states
- Syndrome-style circuit structure
- Measurement results and plots that help interpret the error model

## Suggested Use

Use the bit-flip notebook as the first pass, then open the phase-flip notebook to compare how the two error models differ in circuit structure and outcomes.
