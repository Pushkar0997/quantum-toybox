# Grover's Search

This folder contains small Grover-search examples that show amplitude amplification on 2-qubit and 3-qubit problems.

## Notebooks

- [2_qubit_search.ipynb](2_qubit_search.ipynb) demonstrates the search pattern on a minimal 2-qubit example.
- [3_qubit_search.ipynb](3_qubit_search.ipynb) extends the same approach to a larger search space.

## What to Expect

- Oracle construction for the marked item
- Grover iterations and diffusion steps
- Statevector and histogram-style visualizations of amplitude amplification

## Suggested Use

Compare the 2-qubit and 3-qubit notebooks to see how the search space changes while the algorithmic structure stays the same.
