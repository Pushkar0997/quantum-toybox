# Bell State

This folder covers the simplest entanglement example in the project: preparing a Bell state and inspecting it from a few different angles.

## Notebooks

- [basic_qiskit_implementation.ipynb](basic_qiskit_implementation.ipynb) builds a Bell-state circuit with core Qiskit primitives, then checks the expected measurement behavior and visual output.
- [visual_statevector_tests.ipynb](visual_statevector_tests.ipynb) focuses on the statevector representation, amplitude structure, and visual inspection of the entangled state.

## What to Expect

- Creation of a two-qubit entangled state
- Statevector and probability-style visual checks
- Simple outputs that confirm the Bell-state correlations

## Suggested Use

Start with the basic implementation notebook, then open the visualization notebook to see the same state from the statevector perspective.
