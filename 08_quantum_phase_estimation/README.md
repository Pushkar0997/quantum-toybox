# Quantum Phase Estimation

This folder contains a compact phase-estimation example using a T-gate as the target unitary.

## Notebook

- [t_gate_estimation.ipynb](t_gate_estimation.ipynb) sets up the counting register, applies phase estimation, and visualizes the extracted phase information.

## What to Expect

- Counting-register preparation
- Controlled-unitary structure around a T-gate example
- Measurement results that approximate the encoded phase

## Suggested Use

Open the notebook top to bottom and pay attention to the relationship between the number of counting qubits and the precision of the estimate.
