# Quantum Fourier Transform

This folder explores the Quantum Fourier Transform from both the forward and inverse directions.

## Notebooks

- [standard_qft.ipynb](standard_qft.ipynb) walks through the forward QFT and checks the transformed state.
- [inverse_qft_tests.ipynb](inverse_qft_tests.ipynb) tests the inverse transform and verifies that it recovers the original state.

## What to Expect

- Circuit construction for QFT and inverse QFT
- Statevector inspection after transformation
- Visual checks that confirm the transformation behaves as expected

## Suggested Use

Run the standard QFT notebook first, then open the inverse-QFT notebook to verify the round-trip behavior.
