# Superposition Explorer

This folder explores how to build uniform superpositions and how the behavior changes as the number of qubits increases.

## Notebooks

- [uniform_distribution.ipynb](uniform_distribution.ipynb) prepares a uniform superposition and inspects the resulting distribution.
- [n_qubit_scaling_tests.ipynb](n_qubit_scaling_tests.ipynb) checks how the same idea scales as qubit count grows.

## What to Expect

- Hadamard-based superposition preparation
- Probability distribution visualizations
- Small scaling experiments across multiple qubit counts

## Suggested Use

Open the uniform distribution notebook first, then use the scaling notebook to compare how the circuit behaves at larger sizes.
