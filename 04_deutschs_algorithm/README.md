# Deutsch's Algorithm

This folder contains paired experiments that compare the constant and balanced cases of Deutsch's algorithm.

## Notebooks

- [constant_oracle_test.ipynb](constant_oracle_test.ipynb) constructs a constant oracle and verifies the algorithm output.
- [balanced_oracle_test.ipynb](balanced_oracle_test.ipynb) constructs a balanced oracle and checks the contrasting result.

## What to Expect

- Oracle construction for the two problem classes
- Circuit execution with statevector-based sampling
- Measurement counts and plots that distinguish the cases

## Suggested Use

Run both notebooks and compare their final measurement outcomes to see how Deutsch's algorithm separates constant from balanced functions.
