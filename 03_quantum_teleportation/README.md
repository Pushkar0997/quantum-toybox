# Quantum Teleportation

This folder contains a single notebook that walks through the quantum teleportation protocol end to end.

## Notebook

- [teleportation_protocol.ipynb](teleportation_protocol.ipynb) builds the teleportation circuit, applies the required conditional operations, and shows the recovered state on the target qubit.

## What to Expect

- Preparation of an input state to be teleported
- Entanglement between sender and receiver qubits
- Measurement-driven correction steps
- Output that demonstrates the state transfer

## Suggested Use

Read the markdown cells in sequence and run the notebook top to bottom, since each stage depends on the previous one.
