# quantum-toybox

A small collection of quantum-computing notebooks built with Qiskit.

## Setup

1. Create a virtual environment.
2. Install the core dependencies from `requirements.txt`.
3. Open the notebooks in VS Code or Jupyter and run them in order.

```bash
pip install -r requirements.txt
```

## Roadmap

- `01_bell_state/` - Bell-state preparation and visual statevector experiments
- `02_superposition_explorer/` - Uniform superposition and scaling tests
- `03_quantum_teleportation/` - Teleportation protocol
- `04_deutschs_algorithm/` - Constant and balanced oracle checks
- `05_bernstein_vazirani/` - Hidden-string experiments
- `06_grovers_search/` - Search experiments for 2 and 3 qubits
- `07_quantum_fourier_transform/` - Standard and inverse QFT notebooks
- `08_quantum_phase_estimation/` - Phase estimation with a T-gate example
- `09_variational_quantum_eigensolver/` - VQE optimizer experiments
- `10_error_correction/` - Bit-flip and phase-flip experiments

## Notes

- Notebook files are used instead of `.py` scripts in this version of the project.
- Each notebook is intended as a starting point for experiments and visualization.
